## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Roshan0917/roshan-portfolio.git
```

Move into the project

```bash
cd roshan-portfolio
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
